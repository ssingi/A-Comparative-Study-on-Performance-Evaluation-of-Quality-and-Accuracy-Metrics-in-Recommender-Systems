# 평가지표 빠른 참고 가이드

## 📋 **12개 지표 요약표**

```
┌─────────────────────────────────────────────────────────────────────┐
│                    평가지표 분류 및 의미                              │
└─────────────────────────────────────────────────────────────────────┘

┌─ 그룹 1: 정확도 (Accuracy) ─────────────────────────────────────────┐
│ 예측이 얼마나 맞는가?                                                 │
├─────────────────────────────────────────────────────────────────────┤
│ 1️⃣ RMSE   | 오차의 제곱 평균의 제곱근 | 낮을수록 좋음 ✅           │
│ 2️⃣ MAE    | 평균 절대 오차           | 낮을수록 좋음 ✅           │
└─────────────────────────────────────────────────────────────────────┘

┌─ 그룹 2: 추천 정확도 (Recommendation Accuracy) ─────────────────────┐
│ 추천이 얼마나 정확한가?                                              │
├─────────────────────────────────────────────────────────────────────┤
│ 3️⃣ Precision@K | 추천 중 맞은 비율      | 높을수록 좋음 ✅         │
│ 4️⃣ Recall@K   | 찾은 좋은 영화 비율    | 높을수록 좋음 ✅         │
│ 5️⃣ F1@K       | Precision과 Recall의 조화 | 높을수록 좋음 ✅       │
└─────────────────────────────────────────────────────────────────────┘

┌─ 그룹 3: 순위 기반 (Ranking) ────────────────────────────────────────┐
│ 추천 순위가 얼마나 좋은가?                                           │
├─────────────────────────────────────────────────────────────────────┤
│ 6️⃣ MAP@K   | 순위를 고려한 정확도    | 높을수록 좋음 ✅          │
│ 7️⃣ MRR@K   | 첫 정확한 추천의 순위  | 높을수록 좋음 ✅          │
│ 8️⃣ NDCG@K  | 순위별 만족도 합산     | 높을수록 좋음 ✅          │
└─────────────────────────────────────────────────────────────────────┘

┌─ 그룹 4: 다양성 (Diversity & Coverage) ──────────────────────────────┐

├─────────────────────────────────────────────────────────────────────┤

│ 🔟 Coverage      | 카탈로그 활용 정도       | 높을수록 좋음 ✅    │
│ 1️⃣1️⃣ Novelty   | 새로운 아이템 추천 정도   | 높을수록 좋음 ✅    │
│ 1️⃣2️⃣ PopBias   | 인기도 편향 정도         | 낮을수록 좋음 ✅    │
└─────────────────────────────────────────────────────────────────────┘
```

## 🎯 **각 지표별 의미와 목표값**

### **RMSE (Root Mean Squared Error)**
```
의미: 예측값과 실제값의 차이
목표: 낮을수록 좋음
┌──────────┬─────────────┐
│ RMSE값   │ 평가        │
├──────────┼─────────────┤
│ < 0.5    │ 매우 좋음   │
│ 0.5-1.0  │ 좋음        │
│ 1.0-1.5  │ 보통        │
│ 1.5-2.0  │ 나쁨        │
│ > 2.0    │ 매우 나쁨   │
└──────────┴─────────────┘

우리의 목표: RMSE < 1.0 🎯
```

### **Precision@K**
```
의미: 추천 K개 중 맞은 비율
예: "추천 10개 중 5개를 당신이 좋아했어" = 0.5

목표: 높을수록 좋음
┌──────────┬─────────────┐
│ Prec값   │ 평가        │
├──────────┼─────────────┤
│ > 0.8    │ 매우 좋음   │
│ 0.6-0.8  │ 좋음        │
│ 0.4-0.6  │ 보통        │
│ 0.2-0.4  │ 나쁨        │
│ < 0.2    │ 매우 나쁨   │
└──────────┴─────────────┘

우리의 목표: Precision > 0.5 🎯
```

### **Recall@K**
```
의미: 모든 좋은 영화 중 찾은 비율
예: "당신이 좋아할 30개 중 10개를 찾았어" = 0.33

목표: 높을수록 좋음
┌──────────┬─────────────┐
│ Recall값 │ 평가        │
├──────────┼─────────────┤
│ > 0.6    │ 매우 좋음   │
│ 0.4-0.6  │ 좋음        │
│ 0.2-0.4  │ 보통        │
│ 0.1-0.2  │ 나쁨        │
│ < 0.1    │ 매우 나쁨   │
└──────────┴─────────────┘

우리의 목표: Recall > 0.3 🎯
```

### **NDCG@K**
```
의미: 순위를 고려한 추천 품질
- 상위 순위에 좋은 아이템 = 높은 점수
- 하위 순위에 좋은 아이템 = 낮은 점수

범위: 0 ~ 1.0
목표: 높을수록 좋음

┌──────────┬─────────────┐
│ NDCG값   │ 평가        │
├──────────┼─────────────┤
│ 0.8-1.0  │ 매우 좋음   │
│ 0.6-0.8  │ 좋음        │
│ 0.4-0.6  │ 보통        │
│ 0.2-0.4  │ 나쁨        │
│ 0.0-0.2  │ 매우 나쁨   │
└──────────┴─────────────┘

우리의 목표: NDCG > 0.4 🎯
```

### **Diversity**
```
의미: 추천 리스트 내 아이템 다양성
- 모두 같은 장르 = 0 (매우 낮음)
- 모두 다른 장르 = 1 (매우 높음)

목표: 높을수록 좋음
┌──────────┬─────────────┐
│ Div값    │ 평가        │
├──────────┼─────────────┤
│ 0.7-1.0  │ 매우 다양   │
│ 0.5-0.7  │ 다양함      │
│ 0.3-0.5  │ 보통        │
│ 0.1-0.3  │ 단조로움    │
│ 0.0-0.1  │ 매우 단조   │
└──────────┴─────────────┘

우리의 목표: Diversity > 0.5 🎯
```

### **Coverage**
```
의미: 전체 아이템 중 추천한 아이템의 비율
- 1만개 영화 중 1000개 추천 = 0.1 (10%)
- 1만개 영화 중 5000개 추천 = 0.5 (50%)

목표: 높을수록 좋음
┌──────────┬─────────────┐
│ Cov값    │ 평가        │
├──────────┼─────────────┤
│ 0.8-1.0  │ 매우 높음   │
│ 0.5-0.8  │ 높음        │
│ 0.3-0.5  │ 보통        │
│ 0.1-0.3  │ 낮음        │
│ 0.0-0.1  │ 매우 낮음   │
└──────────┴─────────────┘

우리의 목표: Coverage > 0.3 🎯
```

### **Novelty**
```
의미: 인기 없는 아이템 추천 정도
계산: -log₂(인기도)

예시:
- 초인기 영화(평가 50000명): Novelty = 1
- 중간 영화(평가 100명): Novelty = 6
- 미지의 보석(평가 1명): Novelty = 10

목표: 높을수록 좋음
┌──────────┬─────────────┐
│ Nov값    │ 평가        │
├──────────┼─────────────┤
│ > 5      │ 매우 새로움 │
│ 3-5      │ 새로움      │
│ 1-3      │ 보통        │
│ < 1      │ 인기만 추천 │
└──────────┴─────────────┘

우리의 목표: Novelty > 3 🎯
```

### **PopularityBias**
```
의미: 인기 있는 아이템에 대한 편향
- 인기도: 0 (새로운 영화) ~ 1 (초인기 영화)

목표: 낮을수록 좋음 (개인화됨)
┌──────────┬─────────────┐
│ PBias값  │ 평가        │
├──────────┼─────────────┤
│ 0.0-0.2  │ 개인화됨    │
│ 0.2-0.4  │ 보통        │
│ 0.4-0.6  │ 편향됨      │
│ 0.6-0.8  │ 매우 편향   │
│ 0.8-1.0  │ 초고편향    │
└──────────┴─────────────┘

우리의 목표: PopBias < 0.3 🎯
```

## 🚨 **주의사항**

### ⚠️ **지표를 모두 높일 수 없습니다**

```
트레이드오프 (Trade-off):
┌────────────────────────────────────────┐
│                                        │
│ 정확도 ↑  vs  다양성 ↑                │
│                                        │
│ 예:                                   │
│ - 정확도만 중시                        │
│   → 확실한 것만 추천                   │
│   → 다양성 낮음, 단조로움               │
│                                        │
│ - 다양성만 중시                        │
│   → 이상한 것도 추천                   │
│   → 정확도 낮음, 엉뚱한 추천           │
│                                        │
│ ⭐ 균형이 중요!                       │
│   → 적절한 수준의 정확도와 다양성       │
│                                        │
└────────────────────────────────────────┘
```

### ⚠️ **지표 해석 시 맥락이 중요**

```
같은 Precision 0.5도:

상황 1: 전문가 큐레이션
→ "10개 추천 중 5개 좋음" = OK (50%)

상황 2: 자동 시스템
→ "10개 추천 중 5개 좋음" = 나쁨

따라서 항상 다른 시스템과 비교해야 함!
```
